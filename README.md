# 🗺️ TRIPDROP

> 내 손안의 즉석 여행 플래너

**2026 한국관광공사 × Kakao 관광데이터 활용 공모전** 출품작

## 서비스 소개

출발지와 목적지를 설정하면, AI가 관광지·식당·카페를 동선 최적화로 자동 배치하고 실시간으로 여행을 함께 이끌어주는 모바일 여행 플래너입니다.

## 핵심 기능

- 🔵 **출발지 설정** — 위치 자동감지 + 출발시간·이동수단 선택
- 📍 **관광지 추천** — 한국관광공사 OpenAPI 기반 맞춤 추천
- 🗺️ **실시간 맵** — 선택할 때마다 핀이 꽂히고 경로 자동 연결
- 🔗 **파이프라인** — 관광지 사이에 식사·카페 슬롯 자유롭게 삽입
- ⏱ **타임라인** — 각 지점 예상 도착·출발 시각 자동 계산
- 🚀 **실시간 가이드** — 여행 시작 후 현재 위치 진행도 추적

## 기술 스택

- React 18 + Vite
- Canvas API (지도 렌더링)
- 한국관광공사 OpenAPI
- 카카오맵 API

## 로컬 실행

```bash
npm install
npm run dev
```
